void SvcSerialCommandInterface::endLZ4Message ( )

Calling this function signals the end of a compressed message. The current txBuffer is lz4 compressed before being base64 encoded and sent. The sent message format is "LZ4 [UNCOMPRESSED_SIZE] [BASE64 encoded LZ4 compressed message]".
