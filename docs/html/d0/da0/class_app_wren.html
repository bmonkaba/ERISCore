Detailed Description

Wren is a scripting language. This class is a proxy which mirrors the AppBaseClass into wren.

The use case is to allow scripting of apps Why? with this applications can be dynamically loaded and executed Need another reason? script storage moves from the precious and limited microcontroller flash to the SD card which gives effectivly limitless program storage space. Beware: The available heap defines the upper boundary of VM size
